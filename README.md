Project Title : AI / ML Q&A Chatbot using Vector Store and Streamlit

Project Overview : 
This project demonstrates the development of an end-to-end AI/ML chatbot system built incrementally across multiple internship tasks. The chatbot uses vector-based semantic search to provide accurate, context-aware responses and is deployed using a Streamlit web interface.
The same base project was enhanced step-by-step to complete all internship tasks, ensuring continuity, reproducibility, and professional implementation.
